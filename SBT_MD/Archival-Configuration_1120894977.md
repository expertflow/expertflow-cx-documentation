# CX Knowledgebase : Archival Configuration

Archival Configuration is used in archival process and is now configurable from Voice Recording Solution. Below is the snapshot of the page.

![image-20250603-134926.png](attachments/1120894977/1121320974.png?width=1087)

**Features:**

  1. **Enable Archival Service:** If this button is disabled, the whole archival process will be disabled




Given that Archival Service is enabled user can select the following:

  2. **SFTP Server IP Address:** Archival Server IP Address.

  3. **Folder Path:** Folder path on the above server where the recordings will be archived.

  4. **User name and Password:** A valid user (with appropriate permissions) and its password.

  5. **Archival Modes:**

     1. **Archive Only:** Recordings will be archived after the specified number of days.

     2. **Copy and Archive:** Recordings will be copied from the main server to the archival server after the specified time.

        1. **Copy Time:** `Time after which the files will be copied to the archival server.`

        2. **Purge Time:** `Number of days after which successfully copied files will be removed from the main server(rec).`
