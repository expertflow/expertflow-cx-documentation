# CX Knowledgebase : Meta - Whatsapp Cloud API - Limitations & Key Highlights

**Media Type**| **WhatsApp Cloud API Supported Formats**| **EFCX Supported Formats**  
---|---|---  
**Audio**| `.aac`, `.amr`, `.mp3`, `.m4a`, `.ogg` (OPUS codec)| `.mp3` only  
**Image**| `.jpeg`, `.png` (max 5MB)| `.jpg`, `.png`  
**Video**| `.mp4`, `.3gp` (H.264 video codec, AAC audio codec; max 16MB)| `.mp4` only  
**Document**| `.pdf`, `.doc`, `.docx`, `.xls`, `.xlsx`, `.ppt`, `.pptx`, `.txt` (max 100MB)| `.pdf`, `.doc`, `.docx`, `.ppt`, `.txt`  
**Sticker**| `.webp` (Static: max 100KB; Animated: max 500KB)| Not supported  
**File**|  Supported as document messages (max 100MB)| Supported as shared link or attachment preview (restrictions apply)  
  
_**For further Constraints Related to Whatsapp Media you can follow**_[ _**Whatsapp Documentation**_](https://developers.facebook.com/docs/whatsapp/cloud-api/reference/media) _**.**_

* * *

### Limitations

  1. **Sticker Message Outbound Restriction**

     * Media type for sticker from WhatsApp is `image/webp`

     * Outbound sticker messages are not supported by the system

  2. **Emoji Support**

     * Emojis are currently **not supported** by the connector

  3. **Interactive/Button Messages from Bot**

     * Only **Button-type** interactive messages are supported

  4. **File Sharing Default Setting**

     * File sharing is **disabled by default**

     * Must be manually enabled via: Unified Admin → AgentDesk → Enable File Sharing → Save
