# CX Knowledgebase : Right to left (RTL) support on Agent Desk

This feature enables agents to render the user interface as per the direction of the specified language within the Agent Desk. 

When a user changes the Agent Desk language to an RTL language (Urdu or Arabic), the layout automatically adapts to ensure proper usability and alignment.  
This change impacts multiple UI components, reversing their alignment, text flow, and control order to match RTL reading patterns.

The following changes occur on the customer conversation view when an RTL language is selected:

**Component**| **Description of change**  
---|---  
Customer Info Bar| Moves from the right side of the screen to the left.  
Conversation List| This sidebar moves from the left side of the screen to the right.  
Composer| The composer is also right-aligned.  
Bot Suggestions/ Canned messages| The layout, including action buttons, is changed to RTL.  
  
The following RTL languages are supported at the moment :

  * Arabic

  * Urdu
