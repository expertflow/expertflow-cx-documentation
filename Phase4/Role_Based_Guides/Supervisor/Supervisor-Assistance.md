# Supervisor Assistance

Supervisors in ExpertFlow CX have a comprehensive suite of dashboards and tools to monitor agent performance, ensure service quality, and manage contact center operations in real time.

## Real-Time Dashboards and Monitoring

Supervisor dashboards provide real-time statistics on teams, queues, and active conversations. When a **negative customer sentiment** is detected, supervisors can review the conversation summary for quick context and then choose to:

- **Start a silent monitoring session** — observe the conversation invisibly.
- **Whisper a suggestion** — send a private message visible only to the agent (chat only).
- **Barge in** — join the conversation actively to directly handle the customer.

See [Silent Monitoring](Silent-Monitoring.md) and [Barge-in and Intervention](Barge-in-and-Intervention.md) for detailed guides.

## Agent Hand-Raise Support

Agents can proactively request supervisor help using the **Hand Raise** option. When an agent raises their hand:

- A notification is sent to the supervisor.
- The supervisor can join the conversation to guide the agent privately or take over.

See [Agent Hand Raise](../Functional_Areas/Digital_Channel_Management/Agent-Hand-Raise.md) for details.

## SLA Alert Configuration

Supervisors can configure **SLA violation alerts** to receive timely notifications when conversations breach response time thresholds. This enables proactive intervention and agent guidance before the customer experience deteriorates.

## AI-Powered Assistance

Under **Supervisor AI Assistance**, supervisors gain access to:

| Capability | Description |
|---|---|
| **Conversation Summaries** | AI-generated summaries to quickly understand the current state of an interaction. |
| **Live Transcripts** | Real-time transcription of voice calls. |
| **Sentiment Analysis** | Customer emotion and sentiment tracking throughout the conversation. |

See [Supervisor AI Assistance](Supervisor-AI-Assistance.md) for the full guide.

## Configuration Tools

Beyond monitoring, supervisors can also configure:

| Tool | Description |
|---|---|
| **Business Calendars** | Define business hours, weekday/weekend schedules, and holidays used for routing and IVR behavior. |
| **IVR Prompts** | Select prompts to play based on the caller's region, dialed number (DN), or time period. |
| **Team Announcements** | Create messages broadcast to all agents in the team. |
