# Steps:

You need to follow the below mentioned steps to import the updated reports.

### **Step 1:**

Download the zip file on your system and unzip it.

![](attachments/2525058/2551594.png?width=680)

### **Step 2:**

Goto the folder "**databases** " and edit the file "**cim_reporting_database.yaml** ".

![](attachments/2525058/2551599.png?width=680)

### **Step 3:**

**** The field **sqlalchemy_uri** is the connection string in the below format. Please **update** this as per your configuration. (Don't update any other field.)

**MySQL Format :**[ mysql+](https://docs.expertflow.com/mssql+pymssql:/Username:Password@ServerAddress:PortNumber\(Default:1433\)/Databasename)mysqldb://Username:Password@ServerAddress:PortNumber/DatabaseName

**MsSQL Format :**[ mssql+](https://docs.expertflow.com/mssql+pymssql:/Username:Password@ServerAddress:PortNumber\(Default:1433\)/Databasename)pymssql://Username:Password@ServerAddress:PortNumber/DatabaseName

![](attachments/2525058/2551607.png?width=680)

### **Step 4:**

Now zip the complete folder.

![](attachments/2525058/2551612.png?width=680)

### **Step 5:**

Now Log in to the Superset Administration console by browsing [https://server-ip:port/](https://server-fqdnauth/) URL.

![](attachments/2525058/2551617.png?width=680)

### **Step 6:**

Goto the dashboards list by using URL [https://server-ip:port](https://server-fqdnauth/)[/dashboard/list](http://192.168.1.182:8088/dashboard/list)

![](attachments/2525058/2551620.png?width=680)

### **Step 7:**

Goto the "Import dashboards".

![](attachments/2525058/2551625.png?width=680)

### **Step 8:**

Select the **latest** zip file (that you have created after updating your connection string) and press "IMPORT".

![](attachments/2525058/2551630.png?width=680)

### **Step 9:**

Everything is up now.
