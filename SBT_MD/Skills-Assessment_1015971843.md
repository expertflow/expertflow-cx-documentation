# CX Knowledgebase : Skills Assessment

**Report Summary**|  Skills Assessment report displays agent or team performance based on evaluation form scores. Supports graphical and tabular views, with filters for date, teams, agents, and questionnaires (Evaluation Form). When multiple agents are selected, the report shows average scores per form.   
---|---  
  
![Team and Agent Skill Report GIF.gif](attachments/1015971843/1015054726.gif?width=1479)

## Report Filters

  * Date/Time - Choose the date to filter the data

  * Team - Can select multiple teams.

  * Agents- Can select multiple agents.

  * Questionnaire (Evaluation Form) - Can select multiple questionnaires.
