### Task Media State

A single task can have multiple medias (Voice, Chat) objects and each will have it’s own state object.

**State**| **Description**  
---|---  
 _STARTED_|  When task is created. It’s transitionary State  
 _QUEUED_|  When task is in queue  
 _RESERVED_|  When task is assigned to agent  
 _ACTIVE_|  When agent has accepted the task  
 _CLOSED_|  When agent has ended the media
