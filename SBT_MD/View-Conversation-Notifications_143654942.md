# CX Knowledgebase : View Conversation Notifications

Allows agents to view conversation notifications in real-time.

On the active conversation, agents receive notifications in real-time whenever a new channel session is started/ended; i.e. the time when the customer initiated the conversation.

In cases of a conferenced/ pull-based chat, the agent can view the notifications whenever a new conversation participant joins/leaves the conversation.

![conversation notifications.png](attachments/143654942/143523879.png?width=566)

When a chat ends, the agent can view the notification with the reason for the chat closure.
