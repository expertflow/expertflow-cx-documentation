# CX Knowledgebase : Rasa-X Configuration

This document covers the setup for activating the RASA-X NLU model.

  1. 
[code]Open http://machine-ip:30800 in the browser and login into 
[/code]

![](attachments/2531579/2551174.png?width=587)
  2. Login into rasa admin console. The default username and password is 'admin'
  3. Select models option from side navigation bar.
  4. Select the model and click on activate action button.  
![](attachments/2531579/2551179.png?width=587)



  


Please Use the updated model. If the model is not the same as this one, download it in your local computer and upload it in the models tab in rasa-x UI.
