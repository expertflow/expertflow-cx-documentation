# CX Knowledgebase : Campaign Calls Detail Report

**Report Summary**|  Shows the calls attempted by the dialer and not connected with the end user for any reason. The report includes agent-based and IVR-based campaign calls.  
---|---  
  
## Report Columns

Following are the report columns

**Fields**| **Description**  
---|---  
Campaign| Name of the campaign  
DateTime| Date of the row record  
Number Called| The dialed phone number of the customer  
Call Result| Call result of the call as returned by the dialer; i.e., Abandoned, Connected, No Answer, etc.
