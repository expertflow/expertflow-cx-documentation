# CX Knowledgebase : Queue Flushing API

Acts as an API to clear all queues or a specific queue.

When this API is called, all chats in the queue should be cleared/closed with the configurable message to a customer, such as “Your chat has been ended due to technical issues. Please come back later at a convenient time”.   
For more details, see [API for Queue Flushing](https://api.expertflow.com/#194367d0-9c5a-4f80-b6ef-85a004d206d1).
