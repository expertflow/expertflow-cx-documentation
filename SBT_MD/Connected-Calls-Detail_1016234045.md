# CX Knowledgebase : Connected Calls Detail

**Report Summary**|  Shows the details of calls connected. It does not include any sort of abandoned calls.  
---|---  
  
## Report Columns

Folllowing are the report columns

**Fields**| **Description**  
---|---  
Campaign| Name of the campaign  
DateTime| Date of the row record  
Number Called| The dialed phone number of the customer  
Call Duration| Total time duration of the call in seconds  
Agent ID| ID of the agent who handled the call.  
Agent Name| Name of the agent who handled the call.  
Agent Team| Name of the team to which the agent belongs.  
Wrap-up| Displays the Wrap-up pushed by the agent at the end of the call.
