# CX Knowledgebase : WebRTC Detail Report

**Report Summary**|  Shows the details of webrtc link.  
---|---  
  
## Report Columns

The following are the report columns

**Fields**| **Description**  
---|---  
Conversation ID| The unique identifier of a customer conversation  
Agent Name| Name of the agent who participated in the conversation.  
Status| Status of the conversation, **e.g., successful, unsuccessful, state changed, expired, etc.**  
Expiration Reason| Reason for the termination of the WebRTC link,**e.g., successful_call, not_answered, timeout, etc.**  
Channel| Name the channel associated with this conversation.  
Link Generation Time| The date and time when the webrtc link is generated.
