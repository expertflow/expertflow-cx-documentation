# Customer Labels

Customer Labels allow businesses to categorize customers based on characteristics such as customer segment, language, tier (gold, platinum, standard), or any other relevant grouping. Labels created by administrators are available to agents for assignment during live conversations.

## Creating Customer Labels

Labels are managed by administrators from the Unified Admin interface.

1. Select **Customer Labels** from the main menu in the top-left corner.
2. Click the **+** button to create a new label.
3. Fill in the label details:

| Field | Description |
|---|---|
| **Label** | The label name. Labels are case-sensitive. Duplicate label names are not allowed. |
| **Color** | A color for visual identification in the label list. |

4. Click **Save**.

Once created, the label is immediately available to agents during customer conversations.

## Assigning Labels to Customers

Agents can assign labels to customers from the customer profile panel during an active conversation. Labels can also be created dynamically by agents while updating a customer profile — without requiring an administrator to create them in advance.

## Use Cases

| Label Example | Purpose |
|---|---|
| `Gold`, `Platinum`, `Standard` | Customer tier for priority routing or differentiated service |
| `English`, `Arabic`, `French` | Preferred language for agent matching |
| `VIP`, `Enterprise`, `SMB` | Customer segment for reporting and analytics |
| `Churn Risk`, `Upsell Candidate` | CRM-driven flags for targeted agent handling |
