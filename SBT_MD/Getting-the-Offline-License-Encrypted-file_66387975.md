# CX Knowledgebase : Getting the Offline License Encrypted file

  1. Go to the link attached . <https://stage-web.expertflow.com/api-manager/> and you'll be prompted to enter your master key to generate the license file.

  2. Upload your license key and click on Download.  


![sw.png](attachments/66387975/66387996.png?width=516)
  3. Using the downloaded file, you can apply it on Unified Admin in Offline Mode tab in license configurations.
