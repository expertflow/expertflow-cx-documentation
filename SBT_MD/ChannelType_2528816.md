# CX Knowledgebase : ChannelType

Object Name| ChannelType  
---|---  
Description| This object contains the attributes of the channel type object.   
  
  


Parameter| Type| Description  
---|---|---  
`id `REQUIRED``| String| This is the UUID and contains a unique identifier of channel type. Sample format is `a5c80b3f-ea41-4caf-979d-641a1c32f9bd`  
`name `OPTIONAL``| String| Name of the channel e.g. FACEBOOK, VIBER, WHATSAPP, SMS, WEB, CX_VOICE, CISCO_CC, GENERIC, TWITTER, TELEGRAM, INSTAGRAM.  
``channelLogo OPTIONAL``| String|   
  
`isInteractive `OPTIONAL``| Boolean| Returns `TRUE` if the channel is interactive.  
``mediaRoutingDomain ``OPTIONAL| String|
