# CX Knowledgebase : EFCX Reports Configuration (Superset)

This document covers the setup for importing the reports in the Apache Superset 2.0 Instance.

To delete the previous report to import the new one. Follow this document [here.](https://expertflow-docs.atlassian.net/wiki/x/gwDzAg)

Don't import **both** MsSQL & MySQL reports on the same Apache superset server.

**1.** Go to the directory **databases.**

![](attachments/1190953015/1190953085.png?width=680)

**2.** Now open file "cim_reporting_database.yaml".

![](attachments/1190953015/1190953088.png?width=680)

**3.** The field **sqlalchemy_uri** is the connection string in the below format. Please update this as per your configuration. (Don't update any other field.)

**MySQL Format :**[mysql+](https://docs.expertflow.com/mssql+pymssql:/Username:Password@ServerAddress:PortNumber\(Default:1433\)/Databasename)mysqldb://Username:Password@ServerAddress:PortNumber/DatabaseName  
**MsSQL Format :**[mssql+](https://docs.expertflow.com/mssql+pymssql:/Username:Password@ServerAddress:PortNumber\(Default:1433\)/Databasename)pymssql://Username:Password@ServerAddress:PortNumber/DatabaseName

**4.** Now delete the previous zip file & zip the parent folder again.

![](attachments/1190953015/1190953091.png?width=680)

**5.** Log in to the Superset Administration console by browsing  _https://server-ip:port/_ URL.

![](attachments/1190953015/1190953082.png?width=680)

**6.** Goto the "Import dashboards".

![](attachments/1190953015/1190953094.png?width=680)

**7.** Select the zip file and press "IMPORT".

![](attachments/1190953015/1190953097.png?width=680)

**8.** Everything is up now. 

![](attachments/1190953015/1190953100.png?width=680)

**9.** Now set the UTC offset for reports by the following document [UTC Offset - Reports](UTC-Offset---Reports_2526326.html).

## Superset SSL Configuration

  1. Open the `myCert.cert` file in any text editor and copy its value as shown in the below screenshot.  
  
![](attachments/1190953015/1190953076.png?height=250)

  2. Login into Superset as admin then go to the**Database Connection** from the **Settings** and edit the Database Connection next go to the **Advance Tab** and paste the copied MySQL certificate text into the **Root Certificate** and click on Finish.  
  
![](attachments/1190953015/1190953073.png)
