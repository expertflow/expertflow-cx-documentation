# CX Knowledgebase : Tenant Admin Configurations

This is the Contact Center Administrator guide for configuring Expertflow CX.  
  
## Getting Started

### Logging In to Unified Admin

Log in to **Unified Admin** using the administrator user account supplied by Expertflow. You’ll be asked to change your password on your first login. Follow the screen instructions to change the password. 

## Add Language, Timezone Settings

Add all supported languages from Unified Admin to let agents personalize the AgentDesk interface in their preferred language. See [Adding Languages and Time Zones](1568538630.html) to learn how to add system languages and time zones.

## Add Agents, Supervisors, and Teams 

Login to IAM to create agents and supervisors. 

See [_User Management_](User-Management_1602682928.html) to learn how to create agents and supervisors, and how to assign roles/permissions to them. Use the same document to create and manage teams.

See [Manage Teams ](Manage-Teams_1602027529.html)to see how to create and assign teams to agents.
