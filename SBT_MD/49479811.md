# CX Knowledgebase : Delete Report from (Superset 2.0)

This document covers deleting all dashboards, datasets, charts, and database connections from the Apache Superset 2.0 Instance.

  1. To delete the previous report to import the new one. On the Dashboard click on "**Bulk Select"**


![](attachments/49479811/49447046?width=504)

  2. Select all by clicking the selection box near the name and then click on DELETE.




![](attachments/49479811/49447052?width=504)

  3. Now write DELETE in the delete box then click on **DELETE**. It will delete Dashboards make sure to delete all the dashboards.




![](attachments/49479811/49447058?width=504)

  4. Now goto charts delete all the **Charts** with the same process. Make sure to delete all the Charts.




![](attachments/49479811/49447064?width=504)

  5. Now goto datasets delete all the **Datasets** with the same process. Make sure to delete all the Datasets.




![](attachments/49479811/49447070?width=504)

  6. Now delete the database, Goto **Setting** then **Database Connection.** Click on the delete icon.




![](attachments/49479811/49447076?width=504)

  7. Write DELETE in the confirm section then click on **DELETE.**




![](attachments/49479811/49447082?width=504)

  8. This process will delete all dashboards, datasets, charts, and database connections from Superset. You can now import a new package of all CX reports to Superset.
