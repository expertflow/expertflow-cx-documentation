# CX Knowledgebase : End PULL-based Requests

Enables a supervisor to end PULL-based requests.

## To End A PULL-based Request

The supervisor has to go to [Subscribed Lists](https://expertflow-docs.atlassian.net/wiki/spaces/CX/pages/6162050/4.3+Agent+Guide)

On the **Inbox** view of a subscribed list, hover the mouse over the conversation that needs to be ended, and click on the ![](attachments/2528595/2571591.png?width=50) button to end the conversation.

![](attachments/2528595/2571585.png?width=800)

One can only end a conversation if he/she is granted the relevant permissions to do that. Users having supervisor roles by default have these permissions.
