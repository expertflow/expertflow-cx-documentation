# CX Knowledgebase : Viber

Viber provides [support](Omnichannel-Engagement_2529366.html) to send private messages to users. Viber Direct Messages enable businesses to take an interaction with a user as a private conversation via Viber Bot. Users can now enjoy a range of new features, including support for text messages, emojis, images, audios, videos, file messages, contact information sharing, location sharing, URL links, button messages and sticker messages in addition to the delivery notifications as well within the Viber chat platform. This comprehensive set of additions aims to diversify and enrich communication, allowing users to express themselves more creatively and share various types of content seamlessly.
