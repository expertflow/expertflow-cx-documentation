# Right to Left (RTL) Support on Agent Desk

The Expertflow CX Agent Desk supports Right-to-Left (RTL) rendering for languages such as **Arabic** and **Urdu**. This ensures a native and intuitive experience for agents using these languages.

## How It Works
When the Agent Desk language is set to an RTL language, the entire layout automatically adapts. This includes reversing the alignment of text flow, sidebars, and control elements.

## UI Component Changes

| Component | Description of Change |
| :--- | :--- |
| **Customer Info Bar** | Moves from the right side of the screen to the left. |
| **Conversation List** | This sidebar moves from the left side of the screen to the right. |
| **Composer** | The text input area becomes right-aligned. |
| **Bot Suggestions** | Canned messages and action buttons are re-aligned for RTL patterns. |

## Supported Languages
- Arabic
- Urdu

Agents can switch their language settings in the Agent Desk profile menu to activate these layout changes.
