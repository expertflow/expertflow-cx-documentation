# CX Knowledgebase : Agent Desk Localization

The Expertflow CX Agent Desk provides language localization for agents and supervisors. Users can manually change the interface language from a supported list of languages. The interface also supports **Right-to-Left (RTL)** for languages such as Arabic.  
  
The solution is extensible, and support for other languages can be added on-demand.

Customer names in Arabic and other Unicode scripts are fully supported and displayed correctly in the agent UI

The supported languages include:

  * Arabic

  * Bulgarian

  * English

  * French

  * German

  * Italian

  * Spanish

  * Swahili

  * Urdu




For locale settings and language configuration, see [Updating the Language at Agent Desk](Updating-the-Language-at-Agent-Desk_424214601.html).
