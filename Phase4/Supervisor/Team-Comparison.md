# Team Comparison Report

The Team Comparison Report provides a visual and tabular comparison of **team-level evaluation scores** over a selected time range. It helps Quality Managers and supervisors identify performance trends, compare teams, and drill down to specific evaluation forms, sections, and questions.

Data is displayed across multiple time scales: **daily**, **weekly**, **monthly**, and **yearly**.

## Report Filters

| Filter | Description |
|---|---|
| **Date / Time** | Select a date range to scope the report data. |
| **Team** | Select one or multiple teams to compare. |
| **Questionnaire / Evaluation Form** | Select a single evaluation form to compare scores against. |
| **Question Group** | Filter to one or more question groups (sections) within the form. |
| **Question** | Drill down to specific questions for a granular comparison. |

## Use Cases

- Identify which teams consistently score highest or lowest on evaluations.
- Track whether a team's performance improved after coaching interventions.
- Compare how different teams respond to the same evaluation criteria.
- Surface specific question areas where one team underperforms relative to others.
