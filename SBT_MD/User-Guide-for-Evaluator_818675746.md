# CX Knowledgebase : User Guide for Evaluator

As an **Evaluator** , you are responsible for reviewing assigned conversations, completing evaluations using predefined forms, and submitting assessments within deadlines. This guide will help you:  
  
  * [**Access and manage**](https://expertflow-docs.atlassian.net/wiki/spaces/SBT/pages/818675746/User+Guide+for+Evaluator#Reviews) your assigned reviews in the _Reviews_ tab.

  * [**Conduct evaluations**](https://expertflow-docs.atlassian.net/wiki/spaces/SBT/pages/818675746/User+Guide+for+Evaluator#Starting-a-Review) using conversation data and evaluation forms.

  * [**Track deadlines**](https://expertflow-docs.atlassian.net/wiki/spaces/SBT/pages/818675746/User+Guide+for+Evaluator#Reviews) and prioritize tasks based on status (Pending/Completed).

  * [**Stay informed** ](https://expertflow-docs.atlassian.net/wiki/spaces/SBT/pages/818675746/User+Guide+for+Evaluator#Notifications)with notifications for new assignments and reminders.




## Login

The Login Screen is the entry point for accessing the Quality Management solution. 

### Login Screen

The login screen provides a secure interface for users to access the system by entering their credentials. It includes fields for username and password, a "Remember Me" checkbox for saving login details, and a login button to authenticate and grant access.

## Reviews

The **Reviews** tab is the primary workspace for evaluators.

### Review List View

  1. Displays assigned reviews with:

     1. Status: Pending/Completed.

     2. Due Date: Deadline for review completion.

     3. Created At: Timestamp of assignment.

     4. Questionnaire used.

     5. Submitted

     6. Review score


![13 \(3\).png](attachments/818675746/851083483.png?width=894)

### Starting a Review

  1. Steps to start a review:

     1. Click the Start Review icon.

     2. The Conversation View opens.

     3. Fill out the Review Form alongside the conversation data.

     4. Submit the completed form.


![15 \(2\).png](attachments/818675746/851050754.png?width=903)

## Notifications

Notifications alert evaluators about new reviews and deadlines.

  1. New Assignments: Displays recently assigned reviews.

  2. Reminders: Alerts for approaching deadlines.
