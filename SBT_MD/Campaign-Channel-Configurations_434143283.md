# CX Knowledgebase : Campaign Channel Configurations

The Campaigns component doesn't need explicit channel configuration. The action node in the Campaigns flow builder automatically displays all outbound channels configured in the Unified Admin, as shown in the image below. Therefore, channels only need to be set up in the Unified Admin of the relevant deployment.

![image-20240829-094557.png](attachments/434143283/487522372.png?width=621)

For comprehensive configuration guides for various channels in the Unified Admin, please refer to [Channels](Omnichannel-Engagement_2529366.html)
