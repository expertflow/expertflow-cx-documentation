# Developer & Integrator Quick Start

ExpertFlow CX exposes multiple integration surfaces depending on what you are building. Use this page to find your track.

---

## What are you building?

| Track | You want to… | Start here |
|---|---|---|
| **Agent Desk UI** | Embed a CRM panel, inject custom gadgets, or listen to agent desk events | [AgentManager SDK Integration Guide](../../How-to_Guides/Developer_Integrator/AgentManager-SDK-Integration-Guide.md) |
| **Bot / Virtual Agent** | Connect an NLU engine or custom bot to CX routing | [Bot Connector Developer Guide](../../How-to_Guides/Developer_Integrator/Bot-Connector-Developer-Guide.md) |
| **Channel Connector** | Build a new digital channel (e.g. custom messaging platform) | [Channel Connector Developer Guide](../../How-to_Guides/Developer_Integrator/Channel-Connector-Developer-Guide.md) |
| **CRM Integration** | Push/pull interaction data to/from a CRM via postMessage or API | [CRM Connectors](../../How-to_Guides/Developer_Integrator/CRM-Connectors.md) |
| **Reporting & Data** | Query the reporting database or consume interaction events | [Reporting Database Schema Reference](../../How-to_Guides/Developer_Integrator/Reporting-Database-Schema-Reference.md) |
| **Routing & Scheduling** | Extend routing logic or automate scheduling via API | [Routing Engine Developer Guide](../../How-to_Guides/Developer_Integrator/Routing-Engine-Developer-Guide.md) |
