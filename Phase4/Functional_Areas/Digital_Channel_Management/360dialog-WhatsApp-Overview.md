# 360dialog WhatsApp Integration Overview

The **360-connector** allows businesses to integrate WhatsApp as a customer channel by bridging Expertflow CX with the 360dialog WhatsApp API.

## Core Capabilities
- **API Integration**: Connects to both production (`waba.360dialog.io`) and sandbox (`waba-sandbox.360dialog.io`) environments.
- **Multimedia Support**: Send and receive text, images, and documents via WhatsApp.
- **Unified Management**: Configured directly through the Unified Admin interface alongside other channels.
