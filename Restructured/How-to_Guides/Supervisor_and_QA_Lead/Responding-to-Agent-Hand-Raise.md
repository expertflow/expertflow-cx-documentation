# Responding to Agent Hand Raise

When an agent needs help during a live customer conversation, they can raise their hand to notify their supervisors. You receive a real-time notification and can join the conversation discreetly without the customer's knowledge.

## Receiving a Hand Raise Notification

When an agent raises their hand, all primary and secondary supervisors assigned to that agent's team receive a push notification with two options:

| Action | Description |
|---|---|
| **Join** | Enter the conversation in **Whisper mode** — you can communicate with the agent, but the customer cannot hear or see you. |
| **Dismiss** | Remove the notification without joining. |

Notification behaviour:
- Notifications auto-dismiss after **30 seconds** if no action is taken.
- A sound alert plays once for multiple simultaneous hand raise requests and stops when all notifications are cleared.
- Once you join, your notification is cleared. Other supervisors still see the notification and can also join.

## Responding from the Supervisor Dashboard

In the **Ongoing Conversations Detail Dashboard**, an alert icon appears next to any conversation where an agent has raised their hand. From here you can:
- Click the alert icon to join directly in **Whisper mode**.
- Enter in **Silent Monitoring mode** first, then switch to Whisper mode as needed.

Switching to Whisper mode automatically lowers the hand raise — the alert icon is removed from the dashboard and the conversation view.

## Limitations

- Hand Raise is only available for **digital chat channels**. It is not supported over CX Voice.
- If a voice session is active in the conversation, the Hand Raise icon is disabled for the agent even if a request was already placed.
- More than one supervisor can join the same conversation simultaneously — joining does not dismiss other supervisors' notifications.
