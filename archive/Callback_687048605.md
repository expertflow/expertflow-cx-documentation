# CX Knowledgebase : Callback

The customer can be given the option to be called back from:

  * the Website, 

  * a mobile app

  * the IVR

  * A queue instead of waiting for an agent




Expertflow [Forms](https://expertflow-docs.atlassian.net/wiki/spaces/CX/pages/687506552/4.6+Forms) can be used to collect customer’s info and query which can be stored in the [Conversation Data](https://expertflow-docs.atlassian.net/wiki/spaces/CX/pages/687506764/4.6+Conversation+Data). Expertflow **Conversation Flow** and**Scheduler** can be used to define workflow based on the data collected through [Forms](https://expertflow-docs.atlassian.net/wiki/spaces/CX/pages/687506552/4.6+Forms) and contact the customer at the desired time. The agent can check the context of the call by [Conversation Data](https://expertflow-docs.atlassian.net/wiki/spaces/CX/pages/687506764/4.6+Conversation+Data) when the call lands.
