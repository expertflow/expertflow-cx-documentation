# CX Knowledgebase : Agent  Performance Dashboard

Provides real-time visibility into an agent’s performance and activity metrics. 

This helps supervisors and agents monitor productivity, identify inefficiencies, and ensure compliance with SLAs. The panels display Key Performance Indicators (KPIs) related to agent handling time, readiness, occupancy, and conversation outcomes.

## **Key Perf ormance Indicators**

| **Description**  
---|---  
**Total Handle Time**|  Total time spent by the agent on handled conversations.  
**Talk Time**|  Duration spent on chat-based interactions.  
**Total Hold Time**|  Total time conversations were placed on hold.  
**Ready Time**|  The time agent was in Ready state and available for new tasks.  
**Not Ready Time**|  The time agent was unavailable for handling tasks (break, meeting, etc.)  
**Answered Conversations**|  Number of conversations successfully answered by the agent.  
**Consult Conversations**|  Number of consults/transfers initiated and associated with the agent.  
**RONA Conversations** (Ring No Answer)| Number of tasks offered but not answered by the agent.  
**SLA Breach**|  Number of conversations that were not answered within the SLA threshold.  
**Occupancy %**|  Percentage of logged-in time spent handling interactions.
