# CX Knowledgebase : YouTube Connector – Limitations & Key Highlights

* * *  
  
### Limitations

  1. **Top-Level Comment Restriction**

     * Adding new top-level comments from the CX interface is **not supported**.

  2. **Engagement Restrictions**

     * Agents **cannot like, hide, or pin** any comments.

  3. **Reply Structure Limitation**

     * Only **top-level comments and their first-level replies** are supported.

     * **Third-level replies (sub-replies)** are shown at the same level as first-level replies.

  4. **Attachment Support**

     * **media messages** in comments are **not supported**.
