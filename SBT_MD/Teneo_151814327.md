# CX Knowledgebase : Teneo

It is work in progress and will be integrated with Expertflow CX in future release. 

Teneo has 2 major components, Teneo Web Widget and Teneo Bot . 

We have integrated our EFCX with teneo Web Widget and Bot

  * We support all the message type dynamically which is supported by Teneo Bot and Web Widget 

  * All the message communication between Teneo Bot and Teneo Widget is stored in our EFCX as a conversation history 

  * A seamless transfer to the agent to a dyanimc PQ ,based on the response received from Bot 

  * Agent receevie all the conversation history between Bot and Customer
