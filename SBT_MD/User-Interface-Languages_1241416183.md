# CX Knowledgebase : User Interface Languages

The solution supports multiple languages for the application user interface. Applications support RTL languages such as Arabic as well. 

For all of the frontend application, see the following table. 

**Frontend Applications**| **Localization Capability**  
---|---  
**Customer Widget**| [Customer Widget Localization](Customer-Widget-Localization_1241678195.html)  
**Agent Desk**|  For localization of AgentDesk and available supported languages, see [Agent Desk Localization](Agent-Desk-Localization_1241579818.html)  
  
Support for new languages may be added on-demand.
