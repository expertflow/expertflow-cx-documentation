# CX Knowledgebase : Telegram

Expertflow CX can exchange [supported message types](Omnichannel-Engagement_2529366.html) with a Telegram user. Some open issues/limitations are mentioned in [Telegram Connector Limitations](Telegram-Connector-Limitations_2551889.html).

## Getting Started

  1. Register a Telegram application account and a Telegram bot account. See [Telegram Bot Creation Guide](Telegram-Bot-Creation-Guide_2526362.html)

  2. Specify the Telegram `Bot Token` in CX Telegram connector configurations.
