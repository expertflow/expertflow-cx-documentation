# CX Knowledgebase : QM Admin

As a **QM Admin** , you can configure Quality Management notification delivery and thresholds. You can manage how low-scoring evaluation alerts are sent to Quality Managers: either in real-time for individual reviews or as bulk summaries at set intervals, accessible via a dedicated configuration tab.  
  
To assign a user the QM Admin role, see [this guide](/wiki/pages/createpage.action?spaceKey=SBT&title=IAM%20user%20Creation%20guide&linkCreation=true&fromPageId=1150156801).

## Quality Management workflow for QM Admin

  1. **Configure Alert Threshold**



  * Define the performance threshold**** that triggers alerts for low-scoring evaluations. This ensures that Quality Managers are promptly notified of conversations requiring immediate attention.



  2. **Configure Alert Delivery**



  * Manage how low-scoring evaluation alerts are sent to Quality Managers: either in real-time for individual reviews or as bulk summaries at set intervals, accessible via a dedicated configuration tab.
