# CX Knowledgebase : Questions' Summary Report

**Report Summary**|  Provides an overview of how customers responded to each survey question across multiple interactions.  
---|---  
  
This helps businesses assess overall satisfaction, analyze specific survey questions, and track trends over time.

## Report Columns

The following are the report columns.

**Fields**| **Description**  
---|---  
Total Surveys Offered| The total number of surveys presented to customers.  
Total Responses| The number of responses received across all survey questions.  
Survey Questions| 

  * Each question is listed with its type (e.g., NPS, 5-star rating, MCQ) and the total responses received for each option.

  
Response Breakdown| 

  * For each question, the report shows the number of responses for each answer option. For example, in a 5-star rating, it will show how many customers rated 5 stars, 4 stars, etc.

  
NPS Calculation| 

  * For NPS questions, the report calculates the NPS score based on the distribution of responses (promoters, passives, detractors).

  
Average Scores| 

  * Numerical rating questions like NPS and 5-star ratings will display an average score for tracking customer satisfaction trends.

  
  
## Report Views

The report allows businesses to track response patterns over time, including **daily** , **weekly** , and **monthly** breakdowns.
