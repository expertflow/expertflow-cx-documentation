# CX Knowledgebase : Compatibility Guide-Media Recording

### **OS Compatibility**

We require the customer/partner to install the following software on the server.

**Item**| **Notes**  
---|---  
Debian | Debian 12 full version  
Windows (for Screen Recording)| Windows 10 or 11  
  
### **Docker Engine Compatibility**

**Item**| **Notes**  
---|---  
Docker CE| [Docker CE 18+](https://docs.docker.com/engine/install/debian/) and docker compose 2.35.1  
  
### **Phones Compatibility**

Formal testing and QA are executed only with phones listed in the tables below.

**Item**| **Tested Version**  
---|---  
Micro SIP Phone| 3.21.3  
Cisco IP Communicator| 8.6.1.13
