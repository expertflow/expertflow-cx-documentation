# CX Knowledgebase : Logout

To log out from the agent desk, the agent and all the MRD states should be NOT_READY.

The agent can logout the AgentDesk by selecting a logout reason defined by the system administratorTo log out, click the profile menu at the top right on the agentdesk and click the **Logout** button. Select one reason while getting logged out. Note that these reasons are defined by the system administrator and selecting a logout reason is optional.
