# CX Knowledgebase : Review Screen

The **Review Screen** is a centralized workspace for viewing, accessing, and managing evaluations. It supports both Quality Managers and Evaluators with role-based access to relevant evaluator and allows users to filter, track, and initiate evaluations directly from one place.

#### **1\. Role-Based Interface**

  * **Quality Managers**

Can view all reviews across teams with full metadata including review status, assigned evaluator, evaluation form, created date, due date, agent under review, and score. Managers can filter by evaluation state (Pending, In Progress, Completed) and use the “Assigned to Me” view for personalized task management.

  * **Evaluators**

Have access only to the evaluations assigned to them. Their view displays the minimum required fields for completing evaluations, such as status, due date, evaluation form, and score.




#### **2\. Key Columns in the Review Screen**

Depending on the user role, the following information is displayed:

  * **Status** : Indicates whether the review is Pending, In Progress, or Completed

  * **Due Date** : Deadline for completing the evaluation

  * **Created At** : Timestamp of review creation

  * **Evaluation Form** : Name of the form used for assessment

  * **Reviewer** : Assigned evaluator / Evaluator (visible to managers)

  * **Agent** : Name of the agent being evaluated

  * **Submitted** : Indicates whether the review has been completed

  * **Score** : Final evaluation score after submission




#### **3\. Filtering and Access**

Users can apply filters to organize or locate specific reviews based on status (Pending, Inprogress, Complete) and Evaluation Score. This supports quicker access to pending work and simplifies navigation in high-volume QA environments.

The **“Start Review”** button allows users to initiate evaluations directly. Clicking the button opens the [Conversation View](Conversation-View_822214946.html), where users can analyze the associated interaction and complete the evaluation using the linked questionnaire.
