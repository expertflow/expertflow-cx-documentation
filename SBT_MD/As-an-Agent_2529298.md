# CX Knowledgebase : As an Agent

This document describes the capabilities of the Expertflow Contact Center agents.  
  
## Agent Capabilities

**Capabilities**| **Description**  
---|---  
[**Login to Agent Desk**](Login-to-Agent-Desk_2529062.html)|  The agent can login to Agent Desk using the user's login credentials defined in the Keycloak. When integrated with Cisco Finesse, the agent must specify Cisco login credentials. When the Agent Desk is embedded in Cisco Finesse, the agent is automatically logged in to the Agent Desk.  
[**Change Agent State**](Change-Agent-State_2524603.html)|  Enables agents to change their states globally and MRD-wise.  
[**Accept a Conversation**](Accept-a-Conversation_2527849.html)|  Enables agents to auto-answer, receive a conversation from queues, handle an ongoing conversation, view, update and link customer profiles, view conversation history, and view active channels.  
[**View Conversation Data**](View-Conversation-Data_140869899.html)|  Enables agents to view conversation data for an active conversation and past conversations based on permissions.  
[**Pause/resume a Conversation**](140869657.html)|  Allows agents to pause/resume an active Conversation to temporarily stop SLA and Inactivity timers. The conversation will resume upon the Paused-timer expiry or upon agent request. The Customer may also receive the `Pause`, `Resume` notifications. This may be configured using Conversation Studio.  
[**Consult, Transfer, and Conference**](Consult%2C-Transfer-and-Conference_2528529.html)|  Allows CX agents to consult, transfer an active conversation, or add agents to the conference, and send Whisper Message.  
[**Send a Whisper Message**](Send-a-Whisper-Message_143458310.html)|  Allows agents to send a Whisper message  
[**Send a Message**](Send-a-Message_142868924.html)|  Enables agents to send plain text messages, bot suggestions, formatted text messages, quoted replies, file attachments, receive delivery notifications, view typing indicators, and view and play recording links.  
[**View Conversation Notifications**](View-Conversation-Notifications_143654942.html)|  Allows agents to view conversation notifications in real-time.  
[**Change Agent Desk Language**](Change-Agent-Desk-Language_2529188.html)|  Enables the agent to change the language of the Agent Desk interface.  
[**Raise Hand**](Agent-Hand-Raise_519569565.html)|  This feature enhances agent-supervisor collaboration by allowing agents to request assistance during customer interactions discreetly. Supervisors can promptly identify and respond to these requests without disrupting the conversation flow.  
[**Embed a WebApp in the Agent Desk**](Embed-a-Webapp-in-Agent-Desk_635502601.html)|  Enables agents to switch to third-party applications by simply switching tabs in the **Conversation View**  
[**CTI Call Controls**](CTI-Call-Controls_140869685.html)|  Agents avail with call control functions via Agent Desk (Agent SDK) CTI toolbar. To improve usability, and enhance agents' overall call-handling experience, agents have the option to fix the CTI toolbar in the conversation view.  
[**Handle Voice Recordings**](Handle-Voice-Recording_796590157.html)|  Enables agents to access call recording as part of voice activities and its playback controls, such as seeking and navigating within recordings.  
[**Handling Multi-channel Conversation**](Handle-Multi-channel-Conversation_2528824.html)|  Enables agents to switch channels, to make an outbound contact on a different channel in an ongoing conversation.  
[**Make a Manual Outbound Contact**](Make-a-Manual-Outbound-Contact_2528797.html)|  Allows agents to initiate contact with a customer even if no conversation with the customer is going on.  
[**Apply Wrap-up and Add Notes**](Apply-Wrap-up-and-Add-Notes_2528747.html)|  A wrap-up is a concluding note added to the conversation that helps the business to know what the conversation was about. Expertflow CX agents can provide wrap-ups to conversations that they handle, during or after the conversation.  
[**View Dashboards**](View-Agent-Dashboards_283607081.html)|  The dashboards embedded in Agent Desk are the real-time dashboards for supervisors and agents to view the summary statistics of the contact center including queues, agent states, and the real-time conversations that are ongoing with agents and the bots.  
[**Assign Labels**](Assign-Labels-to-Customers_434667615.html)|  Allows the agent to assign a pre-defined label to a customer.  
[**Understanding Customer List View**](Understanding-Customer-List-View_434471010.html)|  Enables agents to understand the Customer List view and allows agents to set the Column Preferences of their own choice.  
[**Join Pull-based Requests**](Join-Pull-based-Requests_2528122.html)|  Enables agents to pull requests from Pull-based Lists that they are subscribed to and join a conversation.  
[**Leave Conversation**](Leave-Conversation_2529047.html)|  Enables the agent to leave a conversation  
**Browser and sound notifications**|  Enables agents to view browser notifications along with a beep on receiving a new conversation and whenever agents receive new message from the customer. [Browser notifications](Browser-and-Sound-Notifications_2528129.html) should be enabled to receive these notifications. Sound beep on receiving any new message can not be disabled.  
[**Logout**](Logout_2525352.html)|  The agent can logout the AgentDesk by selecting a logout reason defined by the system administrator
