# CX Knowledgebase : How To get Telegram Bot Token

Each bot has a unique token that can be used to control your bot. To get your bot token you must have already created a bot from  [@BotFather](https://t.me/botfather) before.

Don't know what is Telegram Bot Father?? [ Learn more.](https://expertflow-docs.atlassian.net/l/cp/cYBJZV7E)

## Steps to retrieve your bot token

  1. Search for [@BotFather](https://t.me/botfather) and start a new conversation.   


Note

It should be **verified with the tick icon** from the Official telegram.  


Example screenshot:  
  
![](attachments/2526378/2551852.png?height=150)  
  


  2. Send this command _/token_ and you will receive your bot token.
