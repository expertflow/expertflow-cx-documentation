# CX Knowledgebase : AgentPresence

Object Name| AgentPresence  
---|---  
Description| This object contains agent information, agent state, and agent's MRD states.  
  
|   
  
  
  


Parameter| Description  
---|---  
CCUser| The agent object - [CC User](CC-User_2553563.html).   
State| The agent state information  
StateChangeTime| Last time the agent's state was updated.   
AgentMRDStates| Agent's MRD states for all MRDs.
