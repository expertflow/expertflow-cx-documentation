# CX Knowledgebase : Delete Form

This API allows you to delete a form.The endpoints, properties and code examples are given as follows:

**DEL/forms/:formID**

The form ID needs to be passed to delete the particular form:

**Response Code**| **Description**  
---|---  
**200**|  Code 200 implies that the form has been deleted successfully. Example is given as follows:**Response Body:**  

[code] 
     {
        "code":"Deleted",
        "message":"Form Deleted Successfully"
    }
[/code]
