# CX Knowledgebase : Solution API Reference - CX Campaigns - to be reviewed and updated

The following APIs enable efficient management and automation of outbound campaigns, guiding users to schedule, control, and track customer interactions. 

  * [Scheduler APIs](https://api.expertflow.com/#99008c65-bd8d-4ca7-9bbc-ccea900d73c1) : Automate and schedule the execution of outbound campaigns in CX Campaigns

  * [CX Campaigns APIs](https://api.expertflow.com/#f0bc18b4-c41b-4f06-9a24-32f80b39f8df) : Create, modify, and execute personalized campaigns while tracking customer interactions and outcomes.

  * [CX Customers](https://api.expertflow.com/#95ae5c68-621f-417c-93a0-a2dbcea194ca): Upload/ Update Contacts.
