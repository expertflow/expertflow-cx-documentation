# CX Knowledgebase : Evaluator Comparison

**Report Summary**|  The **Evaluator Comparison Report** provides a visual and tabular comparison of how multiple evaluators have scored a single agent using the same evaluation form (questionnaire). It helps identify scoring variations and alignment across evaluators, supporting calibration and consistency in quality assessments. The report supports filtering by date, team, agent, and questionnaire.

## Report Filters

  * Date/Time - Choose the date to filter the data

  * Team - Can select one team.

  * Agents- Can select one agent.

  * Questionnaire (Evaluation Form) - Can select one questionnaire.
