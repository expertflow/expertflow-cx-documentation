# IVR Summary Report

The IVR Summary Report provides a daily aggregated view of IVR activity — showing how customers navigated menus, how many calls were abandoned or transferred, and overall IVR utilization.

---

## Report Columns

| Field | Description |
|---|---|
| **Date** | The date of the summarized data |
| **Language** | The language selection for the summarized data |
| **Menu** | Total number of times customers selected the Main Menu |
| **Sub Menu** | Total number of times customers selected the Sub Menu |
| **Abandoned on IVR** | Total number of calls abandoned while in the IVR |
| **Transfer to Queue** | Total number of calls transferred to a queue from the IVR |
| **Total IVR Duration (HH:MM:SS)** | Sum of all time customers spent in the IVR for the day |
| **Average IVR Duration (S)** | Average time per customer spent in the IVR, in seconds |
| **IVR Utilization (%)** | IVR utilization as a percentage |
| **Maximum IVR Duration (S)** | Longest single IVR session duration, in seconds |

---

## Report Filters

| Filter | Description |
|---|---|
| **Date** | Filter by date |
| **Language** | Filter by customer language selection |
