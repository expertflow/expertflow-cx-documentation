# CX Knowledgebase : Message Body

The message body changes according to the type of message. The types are given as follows:

Title| **Description**  
---|---  
[Action Message](/wiki/spaces/SBT/pages/487227417/Action+Message)| Bot can send Action message to instruct the system to perform a specific action e.g. FIND_AGENT, ASSIGN_AGENT etc. See [this page](https://expertflow-docs.atlassian.net/wiki/spaces/SBT/pages/2527859/Custom+Connector-Bot+Communication) for details.  
[Carousel Message](/wiki/spaces/SBT/pages/486408458/Carousel+Message)| Carousel Message is used to display a slideshow or a group of images.   
[Multimedia Messages](/wiki/spaces/SBT/pages/2525501/Multimedia+Messages)| Media Messages consist of the following types: 

  * Image 
  * Video 
  * Audio
  * File
  * Sticker

  
[URL Message](/wiki/spaces/SBT/pages/2532489/URL+Message)| URL message is specifically used by Agents and Bots (**Outbound Messages**) to send a URL e.g. for app installation, system/license updates.   
[Location Message](/wiki/spaces/SBT/pages/2532513/Location+Message)| Location message is primarily used to send location coordinates to the recipient.   
[Contact Message](/wiki/spaces/SBT/pages/2532495/Contact+Message)| Contact message is primarily used to send details of a contact (name, email, phone etc.) to the recipient.  
[Plain Text Message](/wiki/spaces/SBT/pages/2532510/Plain+Text+Message)| Plain messages are simplest of the message types and are supported by all channels.   
[Receipt Message](/wiki/spaces/SBT/pages/487325734/Receipt+Message)| This message is used to contain the message delivery receipt  
[Email Message](/wiki/spaces/SBT/pages/487227430/Email+Message)| This message is used to send an email.  
[Voice Message](/wiki/spaces/SBT/pages/487227443/Voice+Message)| The user can send audio messages. through this message type  
[Custom Message](/wiki/spaces/SBT/pages/488046598/Custom+Message)| This is a user defined custom message.  
[Template Message](/wiki/spaces/SBT/pages/488013839/Template+Message)| Template Messages are pre-defined messages to be sent to the customer. These messages contain fixed text or fields pre-defined in the system.  
[Button Message](/wiki/spaces/SBT/pages/2532516/Button+Message)| The button message is used to provide pre-defined options to the customer and is specifically for Agents and Conversational Bots (**Outbound Message**). The button API can be customized to send either a single option (as button) or a multiple options (list of buttons).
