# CX Knowledgebase : View Conversation Data

Enables agents to view conversation data for an active conversation and past conversations based on permissions.

To learn more, see [Conversation Data](ConversationData_2526208.html).

By default, right-side panel opens automatically when a conversation is accepted by the agent. Conversation Data pane and Customer info pane will be expanded in this view. Agents can collapse them by clicking on the right-side arrow. The Conversation Data is available in the form of key-value pairs. If any changes are made to this data, they'll be updated here on run time, without requiring a manual refresh.

![conv data.png](attachments/140869899/1235058691.png?width=700)

To see the Conversation Data of past conversations, the agent with certain permissions can only see the data in the Conversation History by clicking on the **Conversation Data** chip shown alongside each conversation. 

![historical conversation data.png](attachments/140869899/141754491.png?width=700)

For now, each update made to the Conversation Data is logged in the conversation history. In future updates, the past conversations will only carry one Conversation Data per conversation, with the latest modified key-value pairs.
