# CX Knowledgebase : onPullModeSubscribedList

Event Name| onPullModeSubscribedList  
---|---  
Event Description| Triggered when the agent unsubscribes or subscribes to the list where certain customer-agent conversations are maintained.  
Emitter| Agent Manager  
  
**Name**| **Description**  
---|---  
name| type: Stringname of the list agent has subscribed  
description| type: Stringdescribes the list  
id| type: Stringsystem generated id of event

    {
            "name": "L1",
            "description": null,
            "id": "6224aa24378c960030df479f"
        }
[/code]
