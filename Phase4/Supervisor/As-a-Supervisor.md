# As a Supervisor

A CX Supervisor has access to all agent capabilities plus additional tools for monitoring, managing, and supporting their agent team. This article provides an overview of the full supervisor capability set.

## Supervisor Capabilities

| Capability | Description |
|---|---|
| **Monitor Dashboards** | Real-time dashboards embedded in Agent Desk showing queue statistics, agent states, and live conversations with agents and bots. |
| **Control Agent Team** | Add agents and supervisors to teams, manage team membership and assignments. |
| **Silent Monitoring** | Join an ongoing conversation as an invisible observer — the agent and customer are unaware of the supervisor's presence. |
| **Barge-in** | Enter a monitored conversation actively when needed to assist the agent or address the customer directly. |
| **Team Announcements** | Create, edit, and delete announcements visible to all agents in the team. |
| **Force Logout Agent** | Forcibly log out any agent regardless of their current state — Ready, Not Ready, in a conversation, in consult, in conference, or handling multiple conversations. Available from the Available Agents Detail Dashboard. |
| **Customer Lists** | View customer records in the Customer List. |
| **Customer Labels** | Create and manage labels to categorize customers by characteristics such as segment, language, or tier (Gold, Premium, Standard). These labels also drive priority routing. |
| **Subscribed Lists** | View subscribed lists and join Pull-based customer requests from a List. |
| **End Pull-based Requests** | Close pull-based requests on a List on behalf of the team. |

## Supervisor-Only Features

The following capabilities are available to supervisors only and are not accessible to standard agents:

- **Team management** — adding and removing agents, supervisors, and teams
- **Silent monitoring and barge-in** — invisible observation and active intervention in conversations
- **Force logout** — logging out agents remotely
- **Team announcements** — broadcast messages to the team
- **Customer labels and priority routing** — configuring label priorities
